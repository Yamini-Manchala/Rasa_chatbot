## Chatbot built using Framework Rasa 

This chat bot will run only with below SW versions using conda environment

Python 3.7.9
rasa==2.0.0
rasa-sdk==2.8.2
spacy==3.1.3
spacy-legacy==3.0.8


#Train the model
rasa train

#Run the actions in another window
rasa run actions

#Run the chatbot
rasa shell
